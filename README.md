# Fastai: From Model to Production

## Prerequisites

Make sure you have pyhton 3.x installed.
The easiest way is by installing [Anaconda](https://anaconda.com/)

## Install Requirement

```
pip install -r requirements.txt
```

## Run Application

```
python main.py
```
